# React-messageboard
A simple message board.
学习react时，写的一个留言板，React+bootstrap
- 使用json文本保存留言
- 使用React实现留言逻辑
- 使用Ajax与服务器响应
- 使用node+express架设服务器和路由

### 本地版本
- 直接打开html
### 服务器版本
- npm install
- node server.js
- 浏览器输入127.0.0.1:3000
